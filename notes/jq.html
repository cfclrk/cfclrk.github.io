Multiline

echo $releases \
    | jq '.[] | select(.name == "v1.16.3").assets[]
              | select(.name == "kops-darwin-amd64").id'

Select

Name contains

aws route53 list-resource-record-sets --hosted-zone-id $id > record_sets
cat record_sets | jq '.ResourceRecordSets[]
    | select(.Name | contains("a string"))
    | .Name'

Name starts with

aws --region us-east-1 lambda list-functions \
    | jq '.Functions[] | select(.FunctionName | startswith("streaming"))'

Select AWS resources with Tag

Use aws –filter

Best way to do this is use AWS server-side filtering. 

aws ec2 describe-addresses --filters "Name=tag:Name,Values=cfclrk"

Period in Key

kubectl get secret $secret -o json \
    | jq -r '.data["ca.crt"]' \
    | base64 --decode

kubectl get secret -o json \
    | jq '.items[]
          | select(.metadata.annotations["kubernetes.io/service-account.name"] == "default")'
