Period in Key

{
    "apiVersion": "v1",
    "data": {
        "tls.crt": "Zm9vCg==",
        "tls.key": "YmFyCg=="
    },
    "kind": "Secret",
    "type": "kubernetes.io/tls"
}

echo $KubeSecret | jq -r '.data["tls.crt"]'

Multiline

echo $GithubRelease \
    | jq '.assets[]
          | select(.name == "py-demo-Darwin-x86_64")
          | .name'

py-demo-Darwin-x86_64

Variable as Argument

ContentType="application/x-mach-binary"
echo $GithubRelease \
    | jq --arg ContentType $ContentType \
         '.assets[] | select(.content_type == $ContentType) | .name'

py-demo-Darwin-x86_64

echo $GithubRelease \
    | jq --arg name "py-demo-Darwin" \
         '.assets[] | select(.name | startswith($name)) | .name'

py-demo-Darwin-x86_64

Name Contains

aws route53 list-resource-record-sets --hosted-zone-id $id > record_sets
cat record_sets | jq '.ResourceRecordSets[]
    | select(.Name | contains("a string"))
    | .Name'

Name starts with

aws --region us-east-1 lambda list-functions \
    | jq '.Functions[] | select(.FunctionName | startswith("streaming"))'

Select AWS resources with Tag

If availabele, use aws --filter

Some AWS commands provide a --filter option that allows filtering by tag. 

aws ec2 describe-addresses --filters "Name=tag:Name,Values=cfclrk"

Sorting

aws s3api list-objects-v2 \
    --bucket $BUCKET \
    --prefix "foo/bar" \
    | jq '.Contents | sort_by(.LastModified)'

If you see an error like Cannot index string with string, it's probably because you are not piping a proper list to it. For example: 

cat $data | jq '.Stuff[] | sort_by(.Time)'  # Bad

cat $data | jq '.Stuff | sort_by(.Time)'    # Good
