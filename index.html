About this Site

This site is just an HTML export of some of my org-mode notes. Org-mode is a literate-programming environment; within Emacs I can execute the code blocks on this site interactively.

The source code for this site is spread out among the following repos: cfclrk/notes, cfclrk/articles, cfclrk/cloudformation, cfclrk/site, cfclrk/cfclrk.github.io, and some org-mode configuration in cfclrk/dotfiles. Properly publishing these org files to HTML has not been easy and it barely works right now. Hopefully, Future-Chris will clean that up!

I used the colors defined in the Doom One theme, which is the theme that my editor uses (I don't use Doom Emacs, I just like the theme). Using the same theme in both places ensures that images look ok in both places.
