Get Info

brew info $formula

Opens the formula's homepage in the browser: 

brew home $formula

View the formula's code. 

brew cat $formula

Show the last 5 git commits that updated this formula: 

brew log gnupg --oneline -n 5

Edit Formula

Opens the formula in $EDITOR. 

brew edit $formula
