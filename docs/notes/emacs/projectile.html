Project Root

In a buffer, do a C-m-: (eval-expression) and run (projectile-project-root). Or: 

(let ((default-directory
        (expand-file-name "~/Projects/py-demo")))
  (projectile-project-root))
#+RESULTS:
/Users/chris.clark/Projects/py-demo
